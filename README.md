## Hajj Management System

System for Management Hajj in KSA
### After install it you must
- create a 6 Item group
   - سكن
   - فيزا
   - نقل جوي
   - نقل بري
   - خدمات أخرى
   - سيارة خاصه

- create a 2 Gender
    - ذكر
    - أنثى

- create a Role List for
    - Hajj Manager
        - has submit for all submittable
        - hide all another workspaces
